# 智能健身教练

- 通过可穿戴设备检测用户心率，运动强度等生理指标
- 根据用户的健身目标（减脂/增肌/提升耐力）动态调整训练计划
- 在用户运动过程中提供实时语音指导和动作纠正
- 评估训练效果并给出饮食建议

PEAS 模型完整描述智能体的任务环境

P 根据用户的生理指标，评估运动表现和饮食习惯，调整训练计划满足用户健身目标
E 可穿戴设备用来检测用户生理指标
A 调用传感器接口，返回生理数据；调用专业的健身网站获取健身规划
S 解析生理指标数据，调整运动计划；实时分析运动强度给出语言指导

# 参考答案

### 指标

可量化，可追踪

- 目标达成率
    - 用户健身目标达成百分比
    - 阶段性体检评估
- 训练计划完成率
    - 系统自动记录训练执行情况
- 生理指标改善情况
    - 可穿戴设备
- 用户留存率
    - 产品后台统计
- 受伤率
    - 用户反馈和异常动作检测
- 用户满意度
    - 问卷评价

### 环境

- 物理环境
    - 健身房，户外等不同运动场景
- 用户本身身体状态
    - 体能，伤病，睡眠等
- 用户目标和时间投入
    - 健身目标，投入时间和金钱成本
- 外部环境
    - 天气，健身房距离
- 历史记录
    - 近期训练饮食

### 执行器

- 实时语言指导
- 规范动作演示视频
- 训练计划调整

### 传感器

- 可穿戴设备
- 智能体脂秤
- 第三方平台 API

### 任务环境特点

- 系统不可能完整知道用户全部身体状态
- 同样的训练计划在不同睡眠、情绪状态下效果不同
- 计划需要有连贯性
