# Agent

### 分类

##### 反射智能体

决策核心规则： `条件-动作`，需要依赖输入，不具备记忆或者预测能力
无法解决状态不足以决策的问题

##### 基于模型的反射智能体

拥有一个内部模型，用于解决状态不满足的条件，通过内部模型智能体有了初步的记忆

##### 基于目标的智能提

##### 基于效用的智能体

对于多个目标进行权衡

##### 学习智能体

包含一个性能元件和一个学习元件，学习元件观察性能元件的执行结果来不断的修正决策策略

### 构成和运行原理

##### 任务环境

- 环境
    - 部分可观测
    - 智能体无法一次性获取足够的信息，需要具备记忆和探索的能力
- 行动结果
    - 可预测
    - 不可预测

##### 运行

不能一次性完成任务，通过持续的循环和环境进行交互

- 感知：通过传感器接收来自环境的输入信息（观察）
- 思考：接收到观察信息，进入决策逻辑（模型内部）
    - 规划
        - 基于观察和记忆，更新对任务和环境的理解，将复杂的目标分解为子任务
    - 工具选择
- 行动

##### 交互

用来规范和环境
